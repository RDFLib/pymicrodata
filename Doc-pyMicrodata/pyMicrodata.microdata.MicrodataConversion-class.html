get_property_value(self, node, context)

source code 

Generate an RDF object, ie, the value of a property. Note that if this element contains an @itemscope, then a recursive call to MicrodataConversion.generate_triples is done and the return value of that method (ie, the subject for the corresponding item) is return as an object.

Otherwise, either URIRefs are created for <a>, <img>, etc, elements, or a Literal; the latter gets a time-related type for the <time> element, and possible numeric types for the @value attribute of the <meter> and <data> elements.

Parameters:
  • node (DOM Node) - the DOM Node for which the property values should be generated
  • context (EvaluationContext) - an instance of an evaluation context
Returns:
an RDF resource (URIRef, BNode, or Literal)
